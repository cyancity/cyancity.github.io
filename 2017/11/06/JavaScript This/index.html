JavaScript This

JavaScript 中函数在执行时，this 总是指向调用该函数的对象，要判断 this 的指向， 其实就是判断 this 所在的函数属于谁。

简单来说

  • 有对象就指向调用该函数的对象
  • 没调用对象就指向全局对象
  • 用new 构造的就指向新对象
  • 通过apply 或call 或 bind来改变this的指向

JavaScript 采用静态作用域规则

​ 但是 function 中的 this 是一个例外。this 并不指向该函数对象本身，也不指向函数的作用域对象。而是在执行过程中动态绑定到调用该函数的对象上，也就是动态作用域

​ 如果在非严格模式下，函数中如果没有明确绑定对象，会默认绑定到全局对象。严格模式下则会报错。
