原字符

正则表达式由两种基本字符类型组成

  • 原义文本字符 (写什么匹配什么)

  • 元字符 (正则中有特殊含义的非字母字符)

    | 字符 | 含义 |
    | :–: | :——–: |
    | \t | 水平制表符 |
    | \v | 垂直制表符 |
    | \n | 换行符 |
    | \r | 回车符 |
    | \0 | 空字符 |
    | \f | 换页符 |
    | \cX | 与X相对应的控制字符 |

预定义类

字符 等价类 含义
. [^\r\n] 除了回车以及换行符外的所有字符
\d [0-9] 匹配数字
\D [^0-9] 匹配非数字
\s [\t\n\x0B\f\r] 匹配空白符,包括 tab,，换行，垂直tab，换页，回车
\S [^\t\n\x0B\f\r] 非空白符
\w [a-zA-Z_0-9] 单词字符( 字母，数字，下划线 )
\W [^a-zA-Z_0-9] 非单词字符
